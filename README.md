# olympus-data

**Machine-published data for the Olympus dashboard** — https://olympus-ascent.netlify.app

This repo holds exactly one artifact: `data.json`, the built data layer the Olympus app
fetches live on every open. It is regenerated by the dashboard's `build.mjs` (which runs
behind integrity gates) and pushed here by automation — the fitness-sync daemon and the
scoring publish ritual. **Do not edit by hand**; hand edits are overwritten by the next
machine push.

This is the same file the live site serves publicly — no additional data is exposed here.
The app deploys separately (only when its code changes); this repo is what lets the numbers
update without a redeploy.
